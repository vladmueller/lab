# Sphinx-SimplePDF

- [Sphinx-SimplePDF — Sphinx-SimplePDF documentation](https://sphinx-simplepdf.readthedocs.io/en/latest/)

## Initial steps

```bash
python3 -m venv venv
```

```bash
source venv/bin/activate
pip install -r requirements.txt
```

```bash
sphinx-quickstart
```

## Build PDF

```bash
make simplepdf
```
